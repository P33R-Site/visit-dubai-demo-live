üé® Theme Integration

The widget automatically detects and follows your website's theme. Here's how to ensure it works:

Method 1: CSS Class (Recommended)

Add a dark or light class to your <html> or <body> element: 

<!-- For dark theme -->
<html class="dark">

<!-- For light theme -->
<html class="light">

Method 2: Data Attribute

Use a data-theme attribute on your root element:

<!-- For dark theme -->
<html data-theme="dark">

<!-- For light theme -->
<html data-theme="light">

Method 3: Automatic Detection

If neither class nor attribute is set, the widget will automatically detect your theme based on:

  • Background color luminance of your page
  • System preference (prefers-color-scheme)

Dynamic Theme Switching

When you toggle themes on your website, the widget automatically updates! Just toggle the class:

// JavaScript to toggle theme
function toggleTheme() {
  const html = document.documentElement;
  if (html.classList.contains('dark')) {
    html.classList.remove('dark');
    html.classList.add('light');
  } else {
    html.classList.remove('light');
    html.classList.add('dark');
  }
  // Widget theme updates automatically!
}
